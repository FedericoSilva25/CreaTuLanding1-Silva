# CreaTuLanding1-Silva

Este es el primer proyecto del curso de ReactJS de CoderHouse. El objetivo de esta entrega era empezar a practicar la estructura de componentes en React y crear la base para una landing page de un e-commerce.

## ¿Qué hice en este proyecto?

- Armé tres componentes principales:
  - `NavBar`: que contiene el logo de la tienda, los enlaces de navegación y el carrito.
  - `CartWidget`: es el ícono del carrito con una cantidad ficticia (por ahora está fijo en 3).
  - `ItemListContainer`: un componente que recibe una prop con un mensaje de bienvenida.

- En el archivo `App.jsx`, importé `NavBar` e `ItemListContainer` para que se rendericen en la app.

- Usé props por primera vez, pasándole un mensaje desde `App.jsx` hacia `ItemListContainer`.

- No usé CSS personalizado, solamente estilos en línea simples para que se vea ordenado.

## ¿Qué aprendí?

Aprendí cómo dividir una aplicación en componentes, cómo importarlos y usarlos, y cómo enviar datos entre ellos usando props. También empecé a entender cómo funciona la estructura de una app en React y cómo se arma la jerarquía entre componentes.
